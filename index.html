🧠 RISC Processor (Verilog VLSI Project)

📌 Overview

32-bit RISC processor implemented in Verilog HDL with full instruction cycle.

⚙️ Features

  • Program Counter
  • Instruction Memory
  • Register File
  • ALU Operations (ADD, SUB, AND, OR)
  • Control Unit

📊 Sample Output

Time=0   PC=00000000 A=10 B=5 Result=15   (ADD)
Time=5   PC=00000004 A=10 B=5 Result=5    (SUB)
Time=15  PC=00000008 A=10 B=5 Result=0    (AND)
Time=25  PC=0000000c A=10 B=5 Result=15   (OR)
