🧠 RISC Processor (Verilog VLSI)

Designed & Simulated using Verilog HDL

📌 Overview

This project implements a 32-bit RISC processor using Verilog HDL. It simulates the complete instruction execution cycle including Fetch, Decode, Execute, and Write-back.

⚙️ Features

  • 32-bit Architecture
  • Program Counter (PC)
  • Instruction Memory
  • Register File (32 registers)
  • ALU Operations: ADD, SUB, AND, OR
  • Control Unit for instruction decoding
  • Waveform Simulation using GTKWave

📊 Sample Output
